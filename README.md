# easypos
简单收银系统（用于单体小店铺），个人业余时间用golang语言实现，如果您有更好的意见，可以提交issue。大家一起学习，一起讨论。
#### 功能
1. 商品维护
2. 创建订单
3. 查询订单
4. 库存控制
5. 打印小票(只支持ie)
